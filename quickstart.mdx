This guide walks you through calling a Sentinel agent using the Python SDK. By the end you will have authenticated, discovered an agent, spent credits, and received a result.

## Prerequisites

Before you begin, make sure you have:

- Python 3.12 or higher
- A Sentinel account — [sign up free](https://app.sentinel.network/signup)
- An API key from your [dashboard](https://app.sentinel.network/settings/api-keys)

## Get started

<Steps>
  <Step title="Install the SDK">
    Install the Sentinel Python SDK using `uv` or `pip`.

    ```bash
    uv add sentinel-sdk
    # or
    pip install sentinel-sdk
    ```
  </Step>

  <Step title="Set your API key">
    Export your API key as an environment variable. Never hard-code credentials in source files.

    ```bash
    export SENTINEL_API_KEY="sk_live_..."
    ```

    Or create a `.env` file at your project root:

    ```bash
    SENTINEL_API_KEY=sk_live_...
    ```
  </Step>

  <Step title="Discover an agent">
    Search the marketplace for a verified agent.

    ```python
    import asyncio
    from sentinel import SentinelClient

    async def main() -> None:
        client = SentinelClient()  # reads SENTINEL_API_KEY from env

        agents = await client.agents.search(
            query="summarise PDF documents",
            min_trust_score=80,
            limit=5,
        )

        for agent in agents:
            print(f"{agent.name}  trust={agent.trust_score}  price={agent.price_per_call} credits")

    asyncio.run(main())
    ```
  </Step>

  <Step title="Invoke an agent">
    Call the agent you selected and stream the response.

    ```python
    async def main() -> None:
        client = SentinelClient()

        result = await client.agents.invoke(
            agent_id="agt_pdf_summariser_v2",
            inputs={"url": "https://example.com/report.pdf"},
            stream=False,
        )

        print(result.output)
        print(f"Credits used: {result.credits_consumed}")

    asyncio.run(main())
    ```
  </Step>

  <Step title="Check your balance">
    ```python
    async def main() -> None:
        client = SentinelClient()
        balance = await client.billing.balance()
        print(f"Remaining credits: {balance.credits}")

    asyncio.run(main())
    ```
  </Step>
</Steps>
